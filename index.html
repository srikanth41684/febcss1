What is CSS?

  1. CSS Stands for Cascading Style Sheet.

  2. CSS discribes how HTML elements are to be displayed on screen, paper, on in other media.

  3. Css is not nessesary CSS is saves lot of work. It can control the layout of multiple web pages all at once.

  4. External stylesheets are stored in CSS files.

CSS can be added to HTML documents in 3 ways:

  • inline - By using the style attributes inside HTML elements
  • Internal - By using a style elements in the head section
  • External - By using a link element to link to an external css file.
  • The most common way to add CSS, is to keep the styles inexternal css files. However, in this tutorial we will use inline and external styles, because this is easier to demonastrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is the CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In CSS, selectors are used to target a specific element or rang of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide rang of selectors available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
